Welcome to Fairfield United Banking Corporation (FUBC)

This Ebanking system helps the bank manager to view a list of all the Customers registered in the system ascending order of the Customers’ last names. It allows the bank manager to add a new Customer into the system. It allows the bank manager to view a list of all the Accounts held in the system sorted in ascending order of the Account Number. It allows the bank manager to add/open a new Account for an existing customer in the system. you can also display the bank’s net liquidity position. This value is computed by taking the sum of the Savings and Checking accounts balances minus the sum of the Loan accounts balances. It represents the total amount of cash expected in the bank’s vault.
